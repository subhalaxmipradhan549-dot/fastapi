# Book Management API

A RESTful API developed using FastAPI and MS SQL Server for managing book records. The project demonstrates CRUD operations, database integration, and backend API development using Python.

## Tech Stack

* Python
* FastAPI
* MS SQL Server
* SQLAlchemy

## Features

* Add new books
* Retrieve all books
* Retrieve a book by ID
* Update book details
* Delete books

## Installation
'''
pip install -r requirements.txt

uvicorn app.main:app --reload


## API Endpoints

* GET /books
* GET /books/{id}
* POST /books
* PUT /books/{id}
* DELETE /books/{id}
